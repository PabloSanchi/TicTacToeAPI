Endpoints

/aimove

This endpoint returns an AI move given an existing board of Tic-Tac-Toe.

How to call: GET tic-tac-toe-api-sigma.vercel.app/aimove

Request body:

{
    "board": [ "X", "", "", "", "X", "", "O", "O", "" ]
}

Response example:

{
    "board": 
        [ "X", "", "", 
        "", "X", "", 
        "O", "O", "O"],
    "move": 8,
    "over": true,
    "tie": false,
    "winner": "O"
    }

Response explanation:

  • "board": The updated board after the AI move has been made.
  • "move": The index of the AI's move in the updated board array.
  • "over": A boolean indicating if the game is over or not.
  • "tie": A boolean indicating if the game ended in a tie.
  • "winner": The winner of the game, if there is one.
